# Woopec

**Learn C# by drawing graphics – starting with turtle graphics.**

A simple and visual way to get started with C# programming.

👉 [Full guide](https://frank.woopec.net/csharp-turtle-graphics/)

---

## What is Woopec?

Woopec is a C# turtle graphics library for beginners.

- Draw shapes and patterns
- Create simple animations
- Work with multiple turtles

---

## Quick Start

```bash
dotnet new install Woopec.Templates
```

Create a new project and run it.

---

## Example

```csharp
public static void TurtleMain()
{
    var turtle = Turtle.Seymour();

    turtle.Forward(100);
    turtle.Right(90);
    turtle.Forward(100);
}
```

---

## Grow beyond turtle graphics

Woopec doesn’t stop at turtle commands.

After learning the basics, you can start working with:
- Figures (structured graphics objects)
- Pens (fine control over drawing)

This gives you a path from simple examples to real graphics programming.

👉 Learn more: [Main Classes](https://frank.woopec.net/woopec_docs/MainClasses.html)
